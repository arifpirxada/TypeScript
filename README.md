# TypeScript repo

## News website

This fetches data from [Newsapi](https://newsapi.org/).

### Stack

`React` `TypeScript` `TanStack Query` `Tailwindcss` `Axios`

### To get started with News website

```bash
git clone git@github.com:arifpirxada/TypeScript.git
```

```bash
cd TypeScript/
cd cd "News website"/
```

```bash
pnpm install
pnpm dev
```
